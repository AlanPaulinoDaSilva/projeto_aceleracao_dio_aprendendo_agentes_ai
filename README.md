Entrega de Projeto de aceleração "DIO CRIANDO UM AGENTE DE AI"

Durante aceleração, foi adquirido conhecimentos de como sanitizar uma planilha .xls utilizando um script .md em uma claude.ai
Após sanitização da claude, a planilha sanitizada foi normalizada
A planilha sanitizada foi carregada no chatgpt e solicitada que fizesse um dasbord.
Foi criado um agente de IA, solicitando as skils necessárias para realizar a tarefa de criar dasbords quando fosse alimentada com as planilhas ja sanitizadas.
